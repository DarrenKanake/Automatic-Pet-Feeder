🐾 Automatic Pet Feeder

This project aims to design and develop an automatic pet feeder that can dispense food to pets at scheduled times without human intervention.

📌 Objective

To create a reliable and efficient system that ensures pets are fed on time, even when the owner is unavailable.

⚙️ Planned Features
Scheduled feeding system
Controlled portion dispensing
Simple user input for setting time
Low food level detection (optional)

🛠️ Tools & Technologies (Planned)
ESP-32 WROOM
Servo motor
Sensors (optional)
Embedded programming (C/Python)
🚧 Project Status

Planning stage – development has not yet started.
